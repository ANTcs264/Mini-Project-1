# 🎮 Adaptive Interactive Storytelling Game

An AI‑powered, browser‑based game that dynamically adapts its story and difficulty based on your choices. The game uses **Machine Learning** to detect your playstyle and tailors the narrative to match your personality.

## ✨ Features

- 🧠 **AI Personality Detection** – The system analyzes your choices and predicts whether you’re aggressive, diplomatic, stealthy, or balanced.
- 📖 **Dynamic Story Branching** – The story changes in real‑time based on your decisions.
- 📊 **Live Player Stats** – Track your actions (fight, diplomatic, stealth, risky, cautious) with cyberpunk‑style progress bars.
- 🎨 **Immersive Gaming UI** – Neon glows, glitch effects, and animated backgrounds.
- 🔁 **Replayability** – Try different playstyles to unlock unique story paths.

## 🛠️ Tech Stack

| Layer          | Technologies |
|----------------|--------------|
| Backend        | Flask, SQLAlchemy, scikit‑learn, joblib |
| Frontend       | HTML5, CSS3, JavaScript (vanilla) |
| Machine Learning | Random Forest Classifier (trained on synthetic data) |
| Database       | SQLite (lightweight, file‑based) |
| Deployment     | Render (backend) + Vercel (frontend) – optional |
